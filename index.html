Player Experience Goals:

This game was created to have players have the expierence that they need to concentrate and in different body parts

Genre:

This is a catch the falling object learning game.

Game play:

The player has very limited option in this game. To start the game, the player must click on the screen play with their mouse. Afterwards, the game will drop an organ number . Then the player has the option of clicking either the left arrow key or the right arrow key to move the nurse character on screen left or right, respectfully.

To make it more fun I would have the game to have more sound effects, like everytime the player choose wrong letter to display a sad sounds, and when he selects the correct one to have a happy song.

Another function that would have been interesting to see is for the Roman numbers to be divided vy V and I, XL, and CM. As the level progreses the game should get harder and harder
